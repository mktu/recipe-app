# LINEチャットのレシピリスト改善

## ステータス
📋 未着手

## 背景・課題

現在、LINEチャットで返ってくるレシピリストがカルーセル形式（横スクロール）になっており、
複数件返ってくると横スクロールが必要で見づらい。

ユーザーとしては、LIFFアプリに遷移せずLINEチャット内である程度完結したい。

## 方針

**縦リスト型の Flex Message** に変更する。
1つのバブルに複数レシピを縦に並べることで、横スクロールなしで一覧できる。

詳細はLIFFで見る前提とし、チャット内では情報を絞る。

## 実装アイデア

### Flex Message レイアウト

```
┌─────────────────────────┐
│ 🔍 3件見つかりました      │
├─────────────────────────┤
│ 🍜 豚バラ大根の煮物       │
│    豚肉・大根・醤油       │  ← タップでLIFF遷移
├─────────────────────────┤
│ 🥘 肉じゃが               │
│    豚肉・じゃがいも       │  ← タップでLIFF遷移
├─────────────────────────┤
│ 🍖 豚の角煮               │
│    豚肉・醤油・砂糖       │  ← タップでLIFF遷移
├─────────────────────────┤
│     📖 一覧をアプリで見る  │
└─────────────────────────┘
```

### 各アイテムの情報

- レシピ名
- 主要食材（2〜3つ程度）
- タップでLIFFのレシピ詳細ページへ遷移

### 件数が多い場合

- チャット内は上位3〜5件に絞る
- 「一覧をアプリで見る」ボタンでLIFF全件表示へ誘導

## 考慮事項

- Flex Message の縦リストは LINE の仕様上1バブルに収める
- 件数の上限（3件？5件？）は要検討
