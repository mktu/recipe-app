# セッション開始

以下の処理を順番に実行してください：

## 1. SESSION.md を読む

`SESSION.md` を読み、現在の状態を把握する。

## 2. 状態サマリーを表示

以下の形式で簡潔にまとめて表示：

```
## 現在の状態

**フェーズ:** [現在のフェーズ]

**直近の完了タスク:**
- [完了タスクを箇条書き]

**次にやること:**
- [優先タスクを箇条書き]

**ブロッカー・注意点:**
[あれば記載、なければ「なし」]
```

## 3. 作業確認

「何から始めますか？」と尋ねる。
