# セッション開始

進捗の正本は **GitHub Issues**、横断的な現在地は **SESSION.md**。両方から立ち上げる。

## 1. 現在地を読む

- `SESSION.md` を読む（現在のフェーズ・横断的な注意点・参照ポインタ）
- `gh issue list --state open` で未完了タスクを取得する（こちらが進捗の正本）

## 2. 状態サマリーを表示

以下の形式で簡潔にまとめて表示：

```
## 現在の状態

**フェーズ:** [SESSION.md の現在のフェーズ]

**open な Issue:**
- [#番号 タイトル を数件]

**横断的な注意点:**
[SESSION.md から、着手内容に関係しそうなものを抜粋。なければ「特になし」]
```

## 3. 作業確認

「何から始めますか？」と尋ねる。
